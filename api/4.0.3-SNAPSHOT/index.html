    reorder (Stream Processor)

    The following code conducts reordering of an out-of-order event stream. This implements the Alpha K-Slack based disorder handling algorithm which was originally
     described in http://dl.acm.org/citation.cfm?doid=2675743.2771828

    Syntax

    akslack:reorder()
